# GrandMed Website

Deployment-ready Next.js website prototype for GrandMed.

## Run locally

1. Install Node.js from https://nodejs.org
2. Open Terminal / PowerShell in this folder
3. Run:

```bash
npm install
npm run dev
```

4. Open http://localhost:3000

## Deploy to Vercel

1. Upload this folder to a GitHub repository.
2. Go to Vercel and import the GitHub repository.
3. Click Deploy.

## Main files

- `app/page.tsx` — homepage
- `app/about/page.tsx` — about page
- `app/projects/page.tsx` — Kingston project page
- `app/partnerships/page.tsx` — healthcare partnerships page
- `app/investors/page.tsx` — investor page
- `app/contact/page.tsx` — contact page
